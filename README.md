# Don Bosco School Website

This project is a static website for Don Bosco School, showcasing information about various educational offerings, including basisschool and secundair education. 

## Project Structure

- `index.html`: The main entry point of the website, featuring a welcome message and links to various secondary school campuses.
- `basisschool/`: Contains HTML files for each basisschool:
  - `school1.html`
  - `school2.html`
  - `school3.html`
  - `school4.html`
  - `school5.html`
  
- `secundair/`: Contains HTML files for each secondary school:
  - `school1.html`
  - `school2.html`
  - `school3.html`
  
- `persbericht.html`: Contains the latest news and press releases related to the schools.
- `finaliteiten.html`: Provides details about the finalities for secondary schools, with placeholders for school-specific information.
- `infomomenten.html`: Provides information about the information moments for schools, with placeholders for school-specific information.
- `partials/`: Contains partial HTML snippets for different campuses:
  - `campus_sint_denijs_westrem.html`
  - `campus_zwijnaarde.html`
  - `campus_sint_pieters.html`
  
- `css/`: Contains stylesheets for the website:
  - `base.css`: Base styles for layout, typography, and responsive design.
  - `index.css`: Specific styles for the index page, including styles for the introduction section.
  
- `js/`: Contains JavaScript for interactive elements:
  - `base.js`: JavaScript for burger menu and dropdown functionality.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. Navigate through the various pages to explore the information provided about the schools.
